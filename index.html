About Visual Studio

Visual Studio Code(VSC For Short) Is An IDE That U Can Write Codes On To Create Websites.




What Is The "Console Screen?

The Console Screen Is A Screen Where Things Can Be Written Through The Code That Only Displays When Open.

To Open Console Screen,Here's What You Have To Do- 1.Right Click Screen 2.Click The "Inspect" Button 3.You Will See On The Top-Right A Few Tabs 4.If It Currently Shows Elements,Click The Console Screen 5. Your Done!
